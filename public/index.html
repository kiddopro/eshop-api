Welcome.

Hello guys and thanks for visiting us, this is a API for e-commerce web page.

Why.

Well, it's just a simple API. Thats a fact. But the thing is, it's not that simple. This API is made for those devs that just want to practice. We all know that the first thing that we do as a programer is a e-commerce webpage. Matter of fact, this API is my project for practice too.
